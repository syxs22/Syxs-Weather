# México Satelital — Clima en Tiempo Real

Visualizador meteorológico interactivo para el sector de la República Mexicana en tiempo real.

---

## Características

* **Imágenes Satelitales:** Alterna entre visualizaciones GeoColor, actividad eléctrica (Rayos) e Infrarrojo térmico.
* **Interactividad:** Soporte para zoom y arrastre de mapa en alta resolución.
* **Atajos Rápidos:** Presiona `R` para restaurar la vista original o la tecla `Espacio` para actualizar los datos.
* **Diseño Responsivo:** Optimizado para pantallas de escritorio, tabletas y móviles.
